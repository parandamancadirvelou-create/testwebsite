# Menu Item Images

Place your menu item images in this folder with the following names:

- `idly.jpg` - Image for Idly
- `poori.jpg` - Image for Poori
- `dosa.jpg` - Image for Dosa
- `vada.jpg` - Image for Vada
- `samosa.jpg` - Image for Samosa

## Image Requirements

- Recommended size: 400x300 pixels or similar aspect ratio
- Format: JPG, PNG, or WebP
- File size: Keep images optimized (under 500KB each for better performance)

## Adding New Images

When adding new menu items through the Manager menu, you can:
1. Add images to this folder
2. Use the image path in the format: `images/your-image-name.jpg`
3. Or use external image URLs if preferred

## Fallback

If an image is not found, the application will display a placeholder with the item name.
